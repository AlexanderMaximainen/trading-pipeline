# Trading Pipeline

A personal data pipeline that processes trading data exported from Avanza and visualizes it in Power BI.

## Purpose

This project was built to practice data engineering skills outside of school, combining Python, SQL server and Power bi into a real end-to-end pipeline.

The pipeline is built for a real use case. A friend who actively trades on Avanza wanted a way to visualize and analyze his trading history. He exports his transaction data as CSV from Avanza, which is automatically picked up by the pipeline and loaded into a Power BI dashboard.

## Tech Stack

- Python
- pandas
- pyodbc
- SQLAlchemy
- SQL Server
- Power BI
